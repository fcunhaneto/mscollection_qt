Ajuda Editar Diretor ou Criador

Como são muito semelhantes mostramo aqui tanto o menu Editar→Filmes Outros→Diretor como Editar→Séries Outros→Criador

Campos para editar diretor ou criador:

Combobox Filme ou Série

Combobox para selecionar o título do filme para editar o diretor ou título da série para editar o criador.

Tabela Diretor ou Criador

Essa tabela possui três colunas “Diretor”, “Ordem” e “Del”, ou “Criador”, “Ordem” e “Del”, conforme seja um filme ou série, na coluna “Diretor” ou “Criador” você possui um Combobox que serve para selecionar um valor existente ou inserir um outro valor como já definido em Regras de uso dos Combobox, a coluna “Ordem” serve para definir a ordem de importância começando com a maior importância em 1 ou seja o diretor ou criador principal e diminuindo de importância conforme a numeração aumenta, atenção não insira um mesmo valor de importância para dois diretores diferentes em um mesmo filme ou série. Também aqui e defina uma coluna “Del” com campos do tipo caixa de seleção para marcar itens que você deseje excluir.

Botões e Teclas

  • Botão Salvar - Tecla de atalho: Ctrl+S
    Para salvar o filme ou série no banco de dados.
  • Botão Limpar - Tecla de atalho: Ctrl+L
    Esvazia todos os campos para iniciar uma nova edição.
  • Botão Ajuda - Tecla de atalho: Ctrl+H
    Mostra a ajuda para essa janela.
  • Botão Sair - Tecla de atalho: Ctrl+Q
    Fecha a janela.
  • Botão + - Tecla de atalho: Ctrl+T
    Para inserir uma linha na tabela Elenco.
