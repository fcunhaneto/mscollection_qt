Ajuda Inserir Filmes e Séries

Regras Para Combobox

Campos comuns a filmes e séries:

Pesquisar no IMDB:

Nesse item clicando no botão SIM aparece um campo onde você pode inserir uma URL (ex: https://www.imdb.com/title/tt2369396/?ref_=nv_sr_1) do IMDB (um site que talvez tenha a maior base de dados sobre filmes, séries, programas de tv e muito mais) referente ao filme ou série e tentaremos preencher os campos Título, Título Original, Ano, Categorias, Diretor ou Criador, Elenco e Poster.

Título:

Para inserir o título do filme ou série.

Título Original:

Para inserir o título original do filme ou série.

Mídia:

Mídia em que se encontra o filme ou série por exemplo: DVD, Blu-Ray. No moment dá instalação já criamos alguns valores sendo eles DVD, Blu-ray, HD (para filmes e séries salvos em seu computador) e Web ( para filmes e séries que se encontrem na internet).

Ano:

O ano do lançamento do filme ou série. Se você quiser pode inserir outro tipo de informação como o ano em que você adquiriu a mídia, o importante e que você mantenha um padrão comum para tratar os campo.

Categoria 1 e Categoria 2:

Você pode inserir até duas categoria por filme sendo a categoria 1 considerada principal e categoria 2 como secundaria. No momento dá instalação já inserimos no banco de dados diversos valores usados pelo IMDB.

Keyword:

Inserir palavras chaves que fazem referência a vários filmes por exemplo se você tem muitos filmes de uma mesma franquia por exemplo "Duro de Matar"' ou filmes de personagens da "Marvel" você também pode inserir essas palavras aqui isso ajudará você em futuras pesquisas.

Poster:

Esse campo normalmente e criado por uma pesquisa no IMDB onde salvamos um thumbnails do filme no diretório poster. Para inserir seus próprios thumbnails use uma imagem de 182x264px ou uma proporção similar coloque a no diretório poster e coloque aqui o caminho absoluto.

Web URL:

A URL caso o filme ou série encontre-se na web.

Resumo:

Para colocar um breve resumo sobre o filme ou série.

Tabela Elenco

Aqui usamos uma tabela com as seguintes colunas “Ator/Atriz”, “Personagem” e “Star” onde cada linha contém um Combobox para inserir um ator ou atriz e outro para seu personagem e um terceiro campo com uma caixa de seleção para marcar se o ator ou atriz é um dos principais do filme ou série, as regras desses Combobox são as mesmas já descritas no inicio.

Para adicionar novas linhas a serem preenchidas basta clicar no botão + ou pressionar as teclas CTRL+T.

Campos somente para filmes

Box:

Para inserir o título/nome de um box que contenha vários filmes.

Diretor:

Para inserir o diretor principal do filme.

Se você necessitar inserir outros diretores depois de salvar o filme vá no menu Editar→Filme Outros→Diretores selecione o filme pelo título e insira ou edite os valores já existentes. Os diretores são classificados por ordem de inserção e assim eles serão mostrados em uma pesquisa.

Tempo:

Para inserir o tempo de duração do filme.

Campos somente para séries:

Criador:

Para inserir o criador principal da série, sendo em tudo semelhante a o campo Diretor de filmes só que para os criadores das séries.

Temporadas:

Serve para inserir o número de temporadas da série.

De preferência ao formato inicio:fim onde inicio é a primeira temporada e fim a última temporada exemplo 1:4, ou, você pode inserir uma lista com os números de temporadas separados por virgulas exemplo: 1, 2, 3, 4.

Botões e Teclas de Atalho Comuns

  • Botão Salvar - Tecla de atalho: Ctrl+S
    Para salvar o filme ou série no banco de dados.
  • Botão Limpar - Tecla de atalho: Ctrl+L
    Esvazia todos os campos para iniciar uma nova edição.
  • Botão Ajuda - Tecla de atalho: Ctrl+H
    Mostra a ajuda para essa janela.
  • Botão Sair - Tecla de atalho: Ctrl+Q
    Fecha a janela.
  • Botão + - Tecla de atalho: Ctrl+T
    Para inserir uma linha na tabela Elenco.

Botões e Teclas de Atalho Especiais

  • Botão Pesquisar no IMDB - Tecla de atalho: CTRL+Alt+I
    Serve para abrir uma janela contento um campo de texto para inserir uma URL do IMDB.
  • Popular Campos - Tecla de atalho: CTRL+Alt+I
    Serve para preencher os campos da janela com os dados do IMDB.
