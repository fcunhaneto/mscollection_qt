Ajuda Editar Box, Keyword, Categoria, Mídia, Ator/Atriz e Personagem

Todos os submenus incluídos aqui tratam de um relacionamento de muitos para um no banco de dados por isso aqui em qualquer das janelas que forem exibidas você só pode selecionar um valor já existente para editar seu valor corrigindo um erro de valor ou erro de gramatica.

O lugar para excluir um item de uma desses valores do banco de dados é em excluir órfãos.

Campos usados nesses menus:

Aqui são mostradas duas janelas diferentes sendo uma contendo um Combobox para selecionar o valor que você deseja editar e outra com uma Caixa de Texto para editar o valor

Botões e Teclas

  • Botão Salvar - Tecla de atalho: Ctrl+S
    Para salvar o filme ou série no banco de dados.
  • Botão Limpar - Tecla de atalho: Ctrl+L
    Esvazia todos os campos para iniciar uma nova edição.
  • Botão Ajuda - Tecla de atalho: Ctrl+H
    Mostra a ajuda para essa janela.
  • Botão Sair - Tecla de atalho: Ctrl+Q
    Fecha a janela.
