Ajuda Excluir Órfãos

Nesse menu todos os submenus abrem uma janela com uma tabela com uma coluna ou duas(caso de Elenco) no mostrando os valores que já não fazem mais referência a nenhum filme ou série e uma coluna com um campo de seleção para ser marcado caso você realmente deseje excluir um desses valores do banco de dados.

Os valores apresentados nas colunas podem ser excluídos sem maiores problemas, porém, você deve prestar muita atenção ao excluir um desse valores porque eles podem voltar a ser necessários exemplo o valor “Fantasia” do campo categorias pode no momento não estar sendo usado por nenhum filme ou série, mas, ele pode muito bem vir a ser usado em novo filme ou série que você inserir.

Atenção pense

Campos Excluir Órfãos:

Tabela Excluir Órfãos

A maioria das tabelas para excluir órfãos apresenta somente duas colunas uma mostrando o valor e outra uma coluna com uma Caixa de Seleção para você marcar os valores que deseja excluir. A exceção é a tabela para elenco que mostra uma coluna a mais exibindo onde as duas primeiras colunas exibem o relacionamento "Ator/Atriz" e "Personagem" e a coluna "DEL" para selecionar os itens a serem excluídos.

Botões e Teclas

  • Botão Excluir - Tecla de atalho: Ctrl+D
    Para excluir os itens selecionados.
  • Botão Selecionar Tudo - Tecla de atalho: Ctrl+F
    Para inserir uma linha na tabela Elenco.
  • Botão Ajuda - Tecla de atalho: Ctrl+H
    Mostra a ajuda para essa janela.
  • Botão Sair - Tecla de atalho: Ctrl+Q
    Fecha a janela.
