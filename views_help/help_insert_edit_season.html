Ajuda Inserir e Editar Temporadas

Os Campos Inserir e Editar Temporada:

Série:

Para CombBox para selecionar a série cuje a temporada você deseja criar.

Temporada:

Campo obrigatório.

Simplesmente o número da temporada por exemplo se for a segunda temporada insira o número 2.

Ano:

O ano do lançamento da temporada. Se você quiser pode inserir outro tipo de informação como o ano em que você adquiriu a mídia, o importante e que você mantenha um padrão comum para tratar os campo.

Mídia:

Mídia em que se encontra a temporada por exemplo: DVD, Blu-Ray. No moment dá instalação já criamos alguns valores sendo eles DVD, Blu-ray, HD (para filmes e séries salvos em seu computador) e Web ( para filmes e séries que se encontrem na internet).

IMDB URL

O site do IMDB possui por série paginas contendo o título e o resumo de todos os episódios por temporada, busque a temporada que deseja coloque a URL aqui é iremos preencher a tabela episódios com esses dados.

Minha Série URL

Semelhante o site do IMDB o site Minha Série (https://www.minhaserie.com.br/) possui paginas com o título e o resumo dos episódios com uma diferença enquanto o IMDB apresenta os episódio de uma temporada em única pagina o Minha Série possui uma pagina para cada episódio, mas, não se preocupe você não precisa inserir aqui a URL de episódio por episódio basta que você insira a URL do último episódio da temporada e vamos preencher a tabela episódios com os dados para todos os episódios por temporada.

Botões IMDB e Minha Série

Esses botões servem respectivamente para iniciar as buscas no IMDB ou em Minha Série conforme a URL escolhida.

Progresso

Esse campo mostra o progresso do download dos episódios.

Tabela Elenco

Tabela só disponível para edição no menu inserir, para reeditar o elenco vá menu Editar→Temporadas→Elenco.

Aqui usamos uma tabela com as seguintes colunas “Ator/Atriz”, “Personagem” e “Star” onde cada linha contém um Combobox para inserir um ator ou atriz e outro para seu personagem e um terceiro campo com uma caixa de seleção para marcar se o ator ou atriz é um dos principais do filme ou série, as regras desses Combobox são as mesmas já descritas no inicio.

Para adicionar novas linhas a serem preenchidas basta clicar no botão + ou pressionar as teclas CTRL+T.

Tabela Episódio

Tabela com três colunas Cód., Título e Resumo onde devem ser inseridos respectivamente:

  • Cos.: Para inserir um código referente ao episódio da seguinte forma um número de no máximo dois dígitos, formatado com um zero a frente quando for o caso, para a temporada exemplo 02 significa a segunda temporada e outro que indica a ordem dos episódios exemplo 08 significa o oitavo episódio da temporada, esses dois números devem ser separados com um ponto ficando no caso dos nossos exemplos da seguinte forma 02.08;
  • Título: o título do episódio;
  • Resumo: um breve resumo do episódio.

Botões e Teclas de Atalho Comuns

  • Botão Salvar - Tecla de atalho: Ctrl+S
    Para salvar a temporada no banco de dados.
  • Botão Limpar - Tecla de atalho: Ctrl+L
    Esvazia todos os campos para iniciar uma nova edição.
  • Botão Ajuda - Tecla de atalho: Ctrl+H
    Mostra a ajuda para essa janela.
  • Botão Sair - Tecla de atalho: Ctrl+Q
    Fecha a janela.
  • Botão + - Tecla de atalho: Ctrl+T
    Para inserir uma linha na tabela Elenco.
  • Botão EXCLUIR - Tecla de atalho: Ctrl+D
    Só disponível no menu editar. Para inserir uma linha na tabela Elenco.
