Regras Para Combobox

Vários campos do banco de dados têm seus valores apresentados dentro de um Combobox(Caixa de Seleção).

Aqui são apresentadas regras para manipular os valores dentro dos Combobox:

  • Para inserir um novo valor basta selecionar o campo vazio do combobox e digitar o novo valor nele, se aparecer um valor já existente continua a digitar o valor desejado depois para ir para o próximo campo pressione Tab ou clique com o mouse no próximo campo que deseja editar.
  • Para buscar um valor existente basta ir digitando o valor que ele aparecerá automaticamente.
  • Atenção você não pode e não deve editar um valor preexistente mesmo que você tente digitar por cima de um valor preexistente nenhuma alteração irá ocorrer dentro do banco de dados. Isso porque esses valores são referencia para vários outros filmes e séries. Se você realmente quiser editar um desses valores no menu Editar são fornecidas opções para isso.
