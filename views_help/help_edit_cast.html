Ajuda Editar Elenco de Filmes, Séries e suas Temporadas

Como são muito semelhantes mostramo aqui tanto o menu Editar→Filmes Outros→Elenco como Editar→Séries Outros→Elenco

Campos para editar elenco:

Combobox Filmes, Séries e suas Temporadas

Combobox para selecionar o título do filme, série e suas temporada no qual você deseja editar o elenco.

Tabela Elenco

Aqui usamos uma tabela, semelhante a usada em inserir filmes ou série, com as seguintes colunas “Ator/Atriz”, “Personagem” e “Star” onde cada linha contém um Combobox para editar um ator ou atriz e outro para seu personagem e um terceiro campo com uma caixa de seleção para marcar se o ator ou atriz é um dos principais do filme ou série, as regras desses Combobox são as mesmas usadas para inserir filmes ou série.

A grande diferença entre as tabelas inserir filmes ou série com essas é a coluna "Del" com uma caixa do tipo seleção para que você marque uma linha cuje o Elenco(“Ator/Atriz”, “Personagem”) você deseja excluir.

Botões e Teclas de Atalho Comuns

  • Botão Salvar - Tecla de atalho: Ctrl+S
    Para salvar o filme ou série no banco de dados.
  • Botão Limpar - Tecla de atalho: Ctrl+L
    Esvazia todos os campos para iniciar uma nova edição.
  • Botão Ajuda - Tecla de atalho: Ctrl+H
    Mostra a ajuda para essa janela.
  • Botão Sair - Tecla de atalho: Ctrl+Q
    Fecha a janela.
  • Botão + - Tecla de atalho: Ctrl+T
    Para inserir uma linha na tabela Elenco.
